# QSIGN 가이드 문서

QSIGN 운영 및 관리를 위한 실무 가이드 모음입니다.

## 📚 가이드 목록

### 1. DELETE-ERROR-POD-GUIDE.md
**에러 Pod 안전 삭제 가이드**

- 에러 상태의 Pod 식별 방법
- 안전한 삭제 절차
- 롤백 및 복구 방법

**사용 시나리오:**
- ArgoCD에서 에러 상태 Pod가 남아있을 때
- 배포 실패 후 정리가 필요할 때
- Pod가 `CrashLoopBackOff`, `Error`, `ImagePullBackOff` 상태일 때

---

### 2. Q-APP-SYNC-GUIDE.md
**Q-APP ArgoCD 동기화 가이드**

- ArgoCD 애플리케이션 동기화 방법
- Sync 정책 설정
- Auto-sync vs Manual-sync

**사용 시나리오:**
- Q-APP 배포 후 상태가 OutOfSync일 때
- 수동으로 동기화가 필요할 때
- Health 상태가 Degraded일 때

---

### 3. Q-SIGN-FIX-GUIDE.md
**Q-SIGN 문제 해결 종합 가이드**

- 일반적인 Q-SIGN 문제 진단
- 단계별 문제 해결 절차
- 검증 방법

**사용 시나리오:**
- Q-SIGN 서비스가 정상 작동하지 않을 때
- API 호출이 실패할 때
- 서명 생성/검증이 작동하지 않을 때

---

## 🎯 가이드 사용 방법

1. **문제 식별**: 현재 상황에 맞는 가이드 선택
2. **가이드 읽기**: 처음부터 끝까지 순서대로 읽기
3. **단계별 실행**: 가이드의 명령어를 하나씩 실행
4. **검증**: 각 단계 후 상태 확인
5. **문서화**: 해결 과정을 기록

---

## ⚠️ 주의사항

- 프로덕션 환경에서는 반드시 백업 후 작업
- 명령어 실행 전 현재 상태 확인
- 로그는 항상 보관
- 변경 사항은 문서화
